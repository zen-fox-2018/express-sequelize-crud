Release 1: Express + Sequelize - CRUD

Student Buatlah aplikasi Express dengan routing seperti di bawah ini. Pisahkan routing dari file app.js dan terpisah dari fitur lain.

    GET /students (menampilkan semua data teacher)
    GET /students/add (menampilkan form input saja)
    POST /students/add (meng-handle input dari form)
    GET /students/edit/:id (menampilkan form data student berdasarkan id)
    POST /students/edit/:id (meng-handle input dari form saat update)
    GET /students/delete/:id (men-delete data berdasarkan id)
